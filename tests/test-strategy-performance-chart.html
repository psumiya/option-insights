Strategy Performance Chart Test

Test Information

This test page validates the Strategy Performance Chart component.

  • Load Sample Data: Shows typical strategy performance with Call and Put P/L
  • Load Mixed P/L Data: Shows strategies with both positive and negative P/L
  • Load Empty Data: Tests empty state display
  • Update Chart: Tests chart update with animation

Expected Behavior:

  • Grouped bars showing Call (blue) and Put (amber) for each strategy
  • Bars grow up from zero line for positive P/L, down for negative P/L
  • Hover tooltips showing strategy, type, P/L, and trade count
  • Legend showing Call and Put colors
  • Responsive resizing when window changes
  • Smooth transitions on data updates
